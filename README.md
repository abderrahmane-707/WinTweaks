# Windows Tweak

## Description
Windows Tweak is a script designed to improve, optimize, and customize Windows 10/11

## Usage
1. **CREATE A SYSTEM RESTORE POINT BEFORE USING THE TOOL!**
2. Right-click on `Tweak.bat` and select **Run as administrator**
3. Choose what you want from the menu

## Note
This tool modifies system and registry settings
Using it without a restore point may result in irreversible changes
I am not responsible for system issues caused by improper use
